## Overview

FriendHub is a native Android social application built with Kotlin, following  **MVVM + Clean Architecture**.
It demonstrates scalable architecture, real-time communication, and modern Android development practices, with a strong focus on clean code, maintainability, and reactive UI state management.

## Key Features

- **User authentication:** Email/Password, Google Sign-In.
- **User profiles:** create, preview, and edit profiles.
- **Friends system:** send requests, accept/reject, manage friends list.
- **Posts feed:** create posts with text and images, like posts, and add comments.
- **Real-time chat:** one-to-one messaging with image and voice support.
- **Image handling:** upload and display profile and post images.
- **UI/UX:** dark mode support, loading & error states, responsive layouts.

 
## Architecture & Tech Stack

- Kotlin
- MVVM & Clean Architecture (Presentation, Domain, Data layers)
- Repository & Use Case patterns
- Kotlin Coroutines & StateFlow for async and reactive state handling
- Hilt for Dependency Injection
- Firebase: Authentication, Firestore (profiles, posts, friends), Realtime Database (chat), Storage (images & voice messages)
- Android Jetpack: ViewModel, Lifecycle-aware components
- XML UI & Material Design
- Gradle (Kotlin DSL)


## Setup (Firebase)
1. Create a Firebase project.
2. Add an Android app with package name: `com.engyh.friendhub`
3. Download `google-services.json`
4. Put it in: `app/google-services.json`
5. Sync Gradle and run.
